# 📄 Rationale Distillation for Interpretable Academic Opinion Recognition in Scientific Literature

This directory corresponds to our journal submission:

> *Rationale Distillation for Interpretable Academic Opinion Recognition in Scientific Literature*  
> Data Analysis and Knowledge Discovery (accepted, forthcoming)

and extends our previous ADMA work:
- 🔗 Paper: [Explain Before Classify: Contrastive Rationale Distillation for Academic Opinion Recognition](https://link.springer.com/chapter/10.1007/978-981-95-3453-1_21) 
- 📄 Dataset README: https://github.com/znoodle63/AOR-RD/blob/main/adma/README.md  


## 🔍 Overview

Compared to the ADMA version, this work introduces the following improvements:

- **Theoretical refinement**:  
  The seven types of academic opinion expression patterns are further clarified and adjusted  

- **Experiments with multiple teacher models**:  
  We conduct experiments using rationales generated by different teacher models, including:  
  - DeepSeek-V3  
  - Gemini-2.0-Flash  
  - GPT-4.1  

- **Multi-task learning framework**:  
  The student model is trained under a multi-task learning framework that jointly learns **labels** and **rationales**.  
  This differs from the earlier **explain-before-classify** setting, where labels were derived from rationales via rule-based parsing.  
  By explicitly learning both tasks, the model reduces its dependence on deterministic rules and improves robustness
